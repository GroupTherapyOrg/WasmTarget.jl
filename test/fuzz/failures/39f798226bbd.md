function instead of nesting it inside the conditional arm that contains the
try — the surrounding if/else structure is dropped entirely (branch test
computed then ignored; both arms execute; normal try-exit br missing →
trailing unreachable executes). Same root for try-nested-in-CATCH shapes
(invalid wasm: regions classified "sequential"). Fix = region-aware try
structurizer: emit the IF first, try_table nested in the arm; support
inner regions whose enter lies inside an outer CATCH range.
