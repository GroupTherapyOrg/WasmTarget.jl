## Analysis
**Root cause: `Dict` with duplicate String keys + a `SubString` value.**
`Dict("" => "", "" => chomp(""))` has duplicate empty keys (collapses to one entry)
and a `chomp("")` value (a `SubString{String}`). Two known-deep ingredients combine:
duplicate-key Dict construction (cf. [[dbac068444b5]]) and SubString materialization
inside Dict (cf. [[05bc422e7ffb]] — `String`/`codeunit` on a SubString in a nested
build). `isempty(Dict(...))` then traps. **Deferred to Part 2** (deep Dict + SubString
codegen).
