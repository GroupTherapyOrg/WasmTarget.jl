## Diagnostic
```
Native Julia raises DivideError/InexactError/BoundsError as CATCHABLE exceptions; WasmTarget lowers these builtin error paths to raw wasm traps (i64.div_s trap, conversion/bounds check → unreachable), which try/catch cannot intercept — the module traps where native returns the catch value. Confirmed catchable: DomainError (explicit Julia throw path, e.g. sqrt(-1.0)). Confirmed uncatchable: div(x,0) [divide by zero trap], Int64(2.5) [InexactError], first(Int64[]) [BoundsError]. Phase 2: route builtin error paths through the same exception mechanism as explicit throws (or guard+throw before the trapping instruction).
```
